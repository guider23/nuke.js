## What It Does

**nuke.js** scans your page (including shadow DOMs & iframes) and removes watermark links, logos, and badges injected by widget providers ‚Äî without breaking functionality.

It runs automatically on load, keeping your UI clean and distraction-free ‚Äî like the widgets were yours all along.

##  Usage

Include the script via CDN:

```html

```

Once included, it will auto-scan and clean supported widgets. No config needed (unless you want it).
